## 使用方式

### 步驟一：安裝 Antigravity
請先安裝 Antigravity 擴充功能：
- [Antigravity 安裝教學](https://drive.google.com/file/d/1CZd48Mxd-isH-bnJg0ydS13zwQ9N-uFu/view?usp=sharing)

### 步驟二：開始使用
1. 點擊上方「線上體驗」連結
2. 選擇需要的工具
3. 填寫相關資訊
4. 點擊「生成提示詞」
5. 複製產生的提示詞到 AI 助手使用
